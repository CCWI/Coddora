Folder structure:

input_data: 
   Contains the room template (base_room.idf) and the weather file for simulation.
   The weather file contains a typical meteorological year (TMY) for Theresienwiese, Munich, Germany. 
   It can be downloaded from:        
    https://climate.onebuilding.org/WMO_Region_6_Europe/DEU_Germany/

output_data:
   Is empty and will store the simulation results files.

ep_data:
   Is used for temporary EnergyPlus files generated during simulation.
